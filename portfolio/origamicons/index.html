Origamicons

Identicons inspired by folded paper.

After a long and slow demise, SHA-1 is finally dead. What can you do with broken cryptographic functions? Create identicons, of course.

The origamicon generator uses SHA-1 hashes to create almost unique pictures that slightly resemble folded paper. It can be accessed via web interface or used offline as a command line tool.
